# A.G.S. Autodemolizioni

Sito web ufficiale di **A.G.S. Autodemolizioni** — demolizione auto gratuita nelle province di Milano e Monza-Brianza.

## Struttura del progetto

```
ags-site/
├── index.html        ← pagina principale
├── style.css         ← tutti gli stili
├── vercel.json       ← configurazione Vercel
├── assets/
│   └── logo.png      ← logo aziendale
└── README.md
```

## Deploy su Vercel

1. Carica questa cartella su **GitHub** come nuovo repository
2. Vai su [vercel.com](https://vercel.com) → **Add New Project**
3. Importa il repository GitHub
4. Framework Preset: **Other** (nessun framework, HTML statico)
5. Root Directory: `/` (lascia come default)
6. Clicca **Deploy**

Il sito sarà live in meno di 1 minuto.
